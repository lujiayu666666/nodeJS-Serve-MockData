# nodeJS-Serve-MockData

## 使用nodeJs起本地服务，编写点单的post请求、get请求接口。
##### 作用是前端可根据后台给的数据格式与参数调用本地服务，node服务将自己mock好的数据传递给前端，进行页面渲染。提高开发效率，与数据的准确性。
